# АИС Расписание ГУУ

Автоматизированная информационная система составления расписания для Государственного Университета Управления.

## Описание
Система позволяет загружать данные о преподавателях, группах, аудиториях и учебных планах, а затем с помощью алгоритмов оптимизации (генетический, имитация отжига, жадный) формировать оптимальное расписание, минимизирующее конфликты и учитывающее различные ограничения.

## Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Убедитесь, что файлы данных находятся в папке `data/`.
4. Запустите приложение: `python app.py`
5. Откройте браузер по адресу `http://127.0.0.1:8050`

## Структура проекта
- `app.py` - основной файл Dash-приложения.
- `assets/` - статические файлы (CSS, изображения).
- `data/` - CSV-файлы с данными.
- `modules/` - Python-модули с логикой (алгоритмы, валидация, загрузка данных).
